História dos computadores

Voltar

A palavra “computador” vem do verbo “computar” que, por sua vez, significa “calcular”. Sendo assim, podemos pensar que a criação de computadores começa na idade antiga, já que a relação de contar já intrigava os homens.

Dessa forma, uma das primeiras máquinas de computar foi o “ábaco”, instrumento mecânico de origem chinesa criado no século V a.C. Assim, ele é considerado o “primeiro computador”, uma espécie de calculadora que realizava operações algébricas.

Depois do ábaco foram criados outros aparelhos, também analógicos, capazes de realizar contagens, como os ossos de napier, a pascalina (primeira calculadora mecânica do mundo), a máquina de Leibniz, o tear automático, a máquina de diferenças, a máquina analítica de Babbage, o tabulador automático, entre outros.

Todos esses dispositivos são resultado de melhorias. Em outras palavras, se um dispositivo falhou em realizar uma determinada tarefa, outro dispositivo foi criado para atender às suas necessidades. Este desenvolvimento não foi um processo tão rápido como é hoje, pois os recursos eram mais limitados na época.

O que são Clients x Server?

Voltar

Clients

Um cliente é um dispositivo ou programa que solicita serviços ou recursos de um servidor. Em um contexto de rede, o cliente é a parte que inicia a comunicação. Pode ser um computador, smartphone, tablet ou qualquer outro dispositivo capaz de se conectar a um servidor.

Exemplo: Quando você usa um navegador da web (cliente) para acessar uma página da web, o navegador faz uma solicitação ao servidor para obter os recursos da página.

Server

Um servidor é um dispositivo ou programa que fornece serviços, recursos ou dados para outros dispositivos chamados clientes. Em uma arquitetura cliente-servidor, o servidor é responsável por processar as solicitações dos clientes e enviar as respostas apropriadas

Exemplo: Um servidor web armazena e fornece páginas da web para os clientes que fazem solicitações. Quando você acessa um site, seu navegador (cliente) envia uma solicitação ao servidor, que então envia os recursos necessários de volta para o navegador exibir.

Como é uma estrutura básica de um HTML?

Voltar

O código HTML organiza o conteúdo de uma página web de maneira estruturada. Ele utiliza elementos como <head>, <body>, <header>, <nav>, <section>, <footer>, <article>, <aside> para criar uma estrutura semântica.

  • <meta charset="UTF-8">: Define a codificação de caracteres para UTF-8, permitindo a exibição adequada de caracteres especiais.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Configura a visualização responsiva em dispositivos móveis, ajustando automaticamente a largura para o tamanho da tela do dispositivo.
  • <header>: Geralmente contém elementos de cabeçalho, como <h1> para o título principal do site.
  • <nav>: Usado para agrupar links de navegação, como menus.
  • <section>: Divide a página em seções lógicas e pode conter vários elementos, como <article> (para conteúdo autônomo) e <aside> (para conteúdo relacionado).
  • <footer>: Inclui informações de rodapé, como direitos autorais ou links para redes sociais.
  • <article>: Um elemento usado para conteúdo independente e autossuficiente que pode ser distribuído e reutilizado separadamente.
  • <aside>: Usado para conteúdo relacionado ao conteúdo circundante, como barras laterais ou notas explicativas.
  • Comentários HTML: Você também pode adicionar comentários ao código HTML usando <!-- comentário aqui -->. Comentários não são exibidos na página, mas são úteis para documentação.
  • Atributos e Valores: Atributos fornecem informações adicionais sobre os elementos e são definidos dentro das tags. Exemplo: <tag atributo="valor">.
  • Links e Navegação: Os links são criados usando a tag <a> e geralmente têm o atributo href que aponta para o destino do link.
  • Listas: As listas ordenadas (<ol>) e não ordenadas (<ul>) podem ser usadas para criar listas numeradas ou com marcadores, respectivamente. Os itens de lista são representados por <li>.

Entendendo a diferença entre listas ordenadas e não ordenadas

Voltar

Lista Ordenada

A lista ordenada é usada para criar uma lista na qual cada item é numerado ou marcado com um símbolo específico <li>. Os elementos da lista ordenada são definidos pela tag (item de lista). Os números ou marcadores são automaticamente gerados pelo navegador.

Lista Não Ordenada

A lista não ordenada é usada para criar uma lista na qual cada item é precedido por um marcador, como um ponto ou um círculo. Assim como nas listas ordenadas, os elementos da lista não ordenada são definidos pela tag <li>.
